# BlemLink (블램링크)

블램링크는 트위터의 링크를 임베드 친화적인 링크(fixupx.com)로 편리하게 변환하고, 미디어 추출 및 다국어 번역 카드를 생성해주는 안드로이드 앱입니다.

### 1. 빠른 변환 (Quick Convert)
자주 쓰는 기능을 모아둔 곳입니다.
*   **기본 변환**: 가장 기본적인 모드입니다. 사진이나 영상이 포함된 트윗은 본문과 함께 첫 번째 미디어가 미리보기로 뜹니다. (동영상은 재생되지 않고 섬네일 이미지만 표시됩니다.) 여러 개의 미디어가 포함된 트윗에서 특정 미디어를 포함한 트윗을 불러오고 싶을 경우, 미디어 가져오기의 기본 모드를 사용하세요.
*   **텍스트 모드**: 이미지나 영상 없이, 트윗의 텍스트 내용만 깔끔하게 가져옵니다. 글만 공유하고 싶을 때 사용합니다.

### 2. 미디어 빌더 (Media Builder)
트윗에 포함된 미디어(사진/영상) 중 원하는 것을 골라내거나, 여러 장을 한 번에 가져올 때 사용합니다.
*   **기본 [본문:O/출처:O/미디어:O]**: 미디어가 여러 개인 트윗에서, 본문과 함께 내가 원하는 미디어가 미리보기로 뜨도록 링크를 생성합니다.
*   **사진 [본문:X/출처:X/미디어:O]**: 트윗 본문이나 출처 없이 '순수 이미지 파일'만 추출합니다. 버튼을 여러 번 눌러 여러 장의 사진 링크를 한꺼번에 생성하고 복사할 수 있습니다. (GIF는 움직이지 않는 정지 이미지로 변환됩니다.)
*   **영상 [본문:O/출처:O/미디어:O]**: 본문과 함께 동영상의 '섬네일 이미지'를 추출합니다. (동영상은 재생되지 않습니다.)
*   **갤러리 [본문:X/출처:O/미디어:O]**: 트윗 본문 없이, '작성자 정보(ID)'와 '선택한 사진'만 카드 형태로 깔끔하게 담아옵니다. 텍스트는 빼고, 출처가 명시된 깔끔한 이미지 카드를 공유하고 싶을 때 적합합니다.

### 3. 다국어 번역 (Translation)
*   트윗의 본문 내용을 선택한 언어(한국어, 영어, 일본어, 중국어)로 자동 번역하여 보여줍니다. 미디어(사진/영상)는 포함되지 않습니다.
*   (한국어 번역은 미디어가 여러 개 포함된 경우 불안정할 수 있습니다. 텍스트만 있는 트윗에 최적화되어 있습니다.)

## 🔒 개인정보 보호 및 보안
*   **서버 미사용**: 본 앱은 외부 서버를 거치지 않으며, 사용자가 공유한 URL을 수정하는 기능만 수행하는 '클라이언트 기반 변환기'입니다.
*   **데이터 비저장**: 사용자의 활동 기록이나 개인정보를 수집, 저장 또는 외부로 전송하지 않습니다.

## 📱 사용 방법
1.  트위터 앱에서 공유하고 싶은 트윗의 **공유하기** 버튼을 누릅니다.
2.  **공유하기 방식 선택**에서 **BlemLink** 앱을 선택합니다. 앱을 즐겨찾기에 등록하면 빠르게 사용할 수 있습니다.
3.  앱이 열리면 원하는 변환 모드를 선택하여 링크를 생성하고 복사합니다.
